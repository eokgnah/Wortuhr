# Wortuhr

* Deutsch
* Bernerdeutsch
* Ungarisch
* Englisch
* ESP8266 D1 Mini
* WS2812 114x LED(W)-Strip (on Pin D1 / IO5)
* NTP
* initial WLAN Manager (192.168.4.1)
* Webinterface
* OTA Firmware Updates http://wortuhr.local/update
* mDNS http://wortuhr.local
