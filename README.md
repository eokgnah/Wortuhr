# Wortuhr

* Deutsch
* Bernerdeutsch
* Ungarisch
* ESP8266 D1 Mini
* WS2812 114x LED-Strip (on Pin D1 / IO5)
* NTP
* initial WLAN Manager (192.168.4.1)
* Webinterface
* OTA Firmware Uodates
* mDNS http://wortuhr.local
